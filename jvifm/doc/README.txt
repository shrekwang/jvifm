    Jvifm is a file manager with vi like key bindings. if you love vi editor, 
 you may love the Jvifm. since the Jvifm is written in java, you have to 
 install JRE environment to run the application. Jvifm requires jre 1.6 or above.
 and jvifm will try to open the file with "gvim" command as default action, so
 it is a good practice to put "gvim" or "gvim.exe" in your path enviroment variable.
